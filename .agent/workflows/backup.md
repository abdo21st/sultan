# نسخ احتياطي لقاعدة البيانات

يستخدم هذا الأمر لإنشاء نسخة احتياطية من قاعدة البيانات.

## الخطوات

1. تصدير قاعدة البيانات (PostgreSQL)

```bash
pg_dump $DATABASE_URL > backup_$(date +%Y%m%d_%H%M%S).sql
```

1. (اختياري) نسخ احتياطي باستخدام Prisma

```bash
npx prisma db pull
```

1. حفظ النسخة الاحتياطية في مكان آمن

```bash
mkdir -p backups
mv backup_*.sql backups/
```

## ملاحظات

- نفذ النسخ الاحتياطي بشكل دوري (يومي/أسبوعي)
- احفظ النسخ في مكان آمن خارج الخادم
- لا ترفع ملفات النسخ الاحتياطي إلى Git
- استخدم Vercel Postgres Backup للإنتاج
