# بناء البرنامج للإنتاج

يستخدم هذا الأمر لبناء نسخة الإنتاج من البرنامج.

## الخطوات

// turbo

1. بناء البرنامج

```bash
npm run build
```

1. (اختياري) تشغيل نسخة الإنتاج محلياً

```bash
npm start
```

## ملاحظات

- البناء يستغرق عدة دقائق
- الملفات المبنية تُحفظ في مجلد `.next`
- Vercel يقوم بالبناء تلقائياً عند الـ push
- استخدم هذا الأمر للتحقق من عدم وجود أخطاء قبل الـ deployment
