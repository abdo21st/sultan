# تنظيف المشروع

يستخدم هذا الأمر لحذف الملفات المؤقتة وإعادة بناء المشروع.

## الخطوات

// turbo-all

1. حذف node_modules و .next

```bash
rm -rf node_modules .next
```

1. حذف ملفات cache

```bash
npm cache clean --force
```

1. إعادة تثبيت المكتبات

```bash
npm install
```

1. إعادة بناء البرنامج

```bash
npm run build
```

## ملاحظات

- استخدم هذا عند مواجهة مشاكل غريبة
- قد يستغرق إعادة التثبيت عدة دقائق
- تأكد من حفظ جميع التغييرات قبل التنظيف
- يحل معظم مشاكل الـ cache
