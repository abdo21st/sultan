# استعادة قاعدة البيانات

يستخدم هذا الأمر لاستعادة قاعدة البيانات من نسخة احتياطية.

## الخطوات

1. استعادة من ملف SQL

```bash
psql $DATABASE_URL < backups/backup_YYYYMMDD_HHMMSS.sql
```

1. (اختياري) إعادة تعيين قاعدة البيانات أولاً

```bash
npx prisma migrate reset --force
```

1. التحقق من البيانات بعد الاستعادة

```bash
npx prisma studio
```

## ملاحظات

- ⚠️ **تحذير**: هذا سيحذف جميع البيانات الحالية
- تأكد من اختيار النسخة الاحتياطية الصحيحة
- اختبر الاستعادة في بيئة تطوير أولاً
- احتفظ بنسخة احتياطية قبل الاستعادة
