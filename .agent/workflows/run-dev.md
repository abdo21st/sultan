# تشغيل البرنامج محلياً

يستخدم هذا الأمر لتشغيل خادم التطوير المحلي لبرنامج سلطان.

## الخطوات

// turbo

1. تشغيل خادم التطوير

```bash
npm run dev
```

1. افتح المتصفح على: <http://localhost:3000>

## ملاحظات

- الخادم سيعمل على المنفذ 3000
- التغييرات في الكود ستظهر تلقائياً (Hot Reload)
- للإيقاف: اضغط Ctrl+C في Terminal
