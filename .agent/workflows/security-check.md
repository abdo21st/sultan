# فحص الأمان

يستخدم هذا الأمر لفحص المكتبات بحثاً عن ثغرات أمنية معروفة.

## الخطوات

// turbo-all

1. فحص الثغرات الأمنية

```bash
npm audit
```

1. إصلاح الثغرات تلقائياً

```bash
npm audit fix
```

1. إصلاح الثغرات الحرجة (قد يكسر التوافق)

```bash
npm audit fix --force
```

1. عرض تقرير مفصل

```bash
npm audit --json
```

## ملاحظات

- نفذ هذا الفحص شهرياً على الأقل
- راجع التغييرات قبل `--force`
- بعض الثغرات قد تتطلب تحديث يدوي
- تابع تنبيهات GitHub Security
