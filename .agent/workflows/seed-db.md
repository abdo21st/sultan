# ملء قاعدة البيانات

يستخدم هذا الأمر لملء قاعدة البيانات ببيانات تجريبية للتطوير.

## الخطوات

1. إنشاء ملف seed (إذا لم يكن موجوداً)

```bash
# إنشاء prisma/seed.ts
```

1. تشغيل seed script

```bash
npx prisma db seed
```

1. التحقق من البيانات

```bash
npx prisma studio
```

## ملاحظات

- استخدم بيانات واقعية ولكن مجهولة
- لا تستخدم seed في الإنتاج
- يمكن استخدامه لإعداد بيئة تطوير جديدة
- مفيد للاختبار والعروض التوضيحية
