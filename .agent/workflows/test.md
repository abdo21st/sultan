# تشغيل الاختبارات

يستخدم هذا الأمر لتشغيل اختبارات البرنامج.

## الخطوات

// turbo

1. تشغيل جميع الاختبارات

```bash
npm test
```

1. تشغيل الاختبارات في وضع المراقبة

```bash
npm test -- --watch
```

1. تشغيل اختبار محدد

```bash
npm test -- <test-name>
```

1. الحصول على تقرير التغطية

```bash
npm test -- --coverage
```

## ملاحظات

- يجب إضافة Jest أو Vitest أولاً
- اكتب اختبارات للوظائف الحرجة
- استهدف تغطية 80% على الأقل
- نفذ الاختبارات قبل كل commit
