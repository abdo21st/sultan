# تحديث المكتبات

يستخدم هذا الأمر لتحديث جميع المكتبات إلى أحدث إصدار.

## الخطوات

1. فحص المكتبات القديمة

```bash
npm outdated
```

1. تحديث المكتبات (minor & patch)

```bash
npm update
```

1. تحديث إلى أحدث إصدار (بما فيها major)

```bash
npx npm-check-updates -u
npm install
```

1. اختبار البرنامج بعد التحديث

```bash
npm run build
npm test
```

## ملاحظات

- اقرأ CHANGELOG قبل تحديث major versions
- اختبر البرنامج جيداً بعد التحديث
- حدّث مكتبة واحدة في كل مرة للمكتبات الكبيرة
- احتفظ بنسخة احتياطية قبل التحديث
